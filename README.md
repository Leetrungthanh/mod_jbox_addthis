## JBox AddThis SmartLayer module for Joomla

[![GitHub license](https://img.shields.io/github/license/jbox-web/mod_jbox_addthis.svg)](https://github.com/jbox-web/mod_jbox_addthis/blob/master/LICENSE)
[![GitHub release](https://img.shields.io/github/release/jbox-web/mod_jbox_addthis.svg)](https://github.com/jbox-web/mod_jbox_addthis/releases/latest)

This little Joomla module will help you with handling JBox AddThis Smartlayer!

It embeds all the needed scripts and code, and let you manage everything just with simple forms ! No code to add : install it and it works, easy ;)

## Features

* Active share buttons
* Active follow buttons
* Allows you to configure Responsive Design parameters
* Allows you to manage differents social networks

More informations on http://www.addthis.com

## Installation

1. Download module files
2. Create a ZIP archive with the files
3. Send to Joomla's extensions manager
4. Finished ! A new module instance is now created in your modules list !
